Echo App
Rapport Économique France

Echo App est un dashboard économique statique en HTML présentant un état des lieux synthétique de l’économie française.

L’application regroupe, sur une seule page, des **indicateurs macroéconomiques, budgétaires, sectoriels et régaliens**, affichés sous forme de :
- cartes KPI
- tableaux de synthèse
- graphiques interactifs (Chart.js)

Aucune installation, aucun backend, aucun framework.  
Un simple navigateur web suffit.

Fichier
- `EcoApp.html` — application complète (HTML + CSS + JavaScript)

Contenu affiché

Le dashboard inclut :

Macroéconomie
- Croissance du PIB
- Chômage
- Inflation
- Dette publique
- Déficit budgétaire
- Charge d’intérêts

Marché du travail
- Demandeurs d’emploi A/B/C
- Variations trimestrielles et annuelles
- Créations nettes d’emplois

### Pouvoir d’achat & tarifs réglementés
- SMIC
- Électricité
- Livret A / LEP
- Frais bancaires
- Gaz

Climat des affaires
- Indices sectoriels (industrie, services, commerce, bâtiment)

Secteurs clés
- Automobile
- Agriculture
- Construction / BTP
- Investissements étrangers

Régalien
- Défense
- Sécurité intérieure
- Numérique & IA

Tourisme

Comparaison Zone Euro
- Croissance
- Chômage

Visualisation
Les graphiques sont générés via Chart.js (CDN) :
- Trajectoire PIB / chômage
- Climat des affaires
- Comparaison européenne
